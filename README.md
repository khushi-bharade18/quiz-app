# Quiz Application

A responsive and interactive quiz application built using React. The application allows users to register, attempt a timed quiz, and view their final score. It focuses on clean UI, user experience, and efficient state management.

## Live Demo Link

https://quiz-app-sepia-nu.vercel.app/

## Features

- User registration with form validation
- Timed quiz with 10 seconds per question
- Automatic question navigation based on timer
- Real-time score tracking
- Restart functionality
- Responsive design for multiple screen sizes
- Local storage integration for user data persistence

## Tech Stack

- Frontend: React.js
- Styling: Tailwind CSS-
- Form Handling: React Hook Form
- Routing: React Router
- Deployment: Vercel
